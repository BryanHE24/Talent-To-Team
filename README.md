# HireFlow

HireFlow is an AI-to-AI recruitment platform.

### Environment Setup
Create a `.env` file based on this template (not created as a standalone file during Phase 1 to respect file budget constraints):

```env
# Frontend
VITE_API_URL=http://localhost:8000

# Backend
OPENAI_API_KEY=your_key
SUPABASE_URL=your_url
SUPABASE_KEY=your_key
```
